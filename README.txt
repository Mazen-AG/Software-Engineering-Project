# Geeks Hub - Coworking Space Website
# Slash - Empowering Local Brands in Egypt

Welcome to the Geeks Hub GitHub repository! Geeks Hub is a coworking space website developed using ReactJS for the front end, Firebase for user authentication, and Firestore as the realtime database.

## Project Overview

Geeks Hub aims to provide a seamless platform for individuals and teams to collaborate and work in a shared coworking space. The website facilitates various features to enhance the coworking experience:

### Key Features

1. User Authentication: Utilizing Firebase Authentication, users can securely sign up, log in, and manage their accounts.

2. Realtime Collaboration: Firestore realtime database ensures instant updates and collaboration among users, enabling a dynamic coworking environment.

3. Booking System: Users can reserve workspace, meeting rooms, and other facilities, enhancing the organization of the coworking space.

4. Event Calendar: Stay updated on upcoming events, workshops, and meetups within Geeks Hub, fostering a sense of community.

5. User Profiles: Customize your profile, showcase your skills, and connect with other members within the Geeks Hub community.

## Getting Started

To run the project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/geeks-hub.git
   ```

2. Install dependencies:
   ```bash
   cd geeks-hub
   npm install
   ```

3. Set up Firebase:
   - Create a Firebase project on the [Firebase Console](https://console.firebase.google.com/).
   - Add a web app to your project and obtain the configuration.
   - Enable Email/Password authentication.
   - Set up Firestore database rules.
   - Replace the Firebase configuration in `src/firebase/firebaseConfig.js` with your own.

4. Run the application:
   ```bash
   npm start
   ```
